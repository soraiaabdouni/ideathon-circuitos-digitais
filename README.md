# 💡 Ideathon-CD | Circuitos Digitais - UNIFESP

Repositório contendo todos os materiais da entrega final do **Projeto Ideathon - Sistema de Alerta Preventivo para Crises Emocionais em Adolescentes**, desenvolvido pelo grupo **Biologic** para a disciplina Circuitos Digitais da Universidade Federal de São Paulo (UNIFESP).

## 📖 Sobre o projeto

O **Ideathon-CD** consiste em uma maratona de desenvolvimento de soluções tecnológicas e inovadoras para atender a um dos Objetivos de Desenvolvimento Sustentável (ODS) da ONU, por meio do desenvolvimento de um protótipo de circuito digital utilizando o **WiredPanda**.

Este repositório reúne toda a documentação e os arquivos referentes ao projeto desenvolvido pela equipe.

---

# 📂 Estrutura do repositório

```
.
├── circuito/              # Arquivos do circuito desenvolvidos no WiredPanda
├── relatorio/             # Relatório final em PDF
├── pitch/                 # Vídeo do Pitch Acadêmico
├── demonstracao/          # Vídeo demonstrando o funcionamento do circuito
└── README.md
```

---

# 🎯 Objetivo

Desenvolver um protótipo de circuito digital capaz de contribuir com os Objetivos de Desenvolvimento Sustentável (ODS 3 e ODS 10) da ONU, utilizando conceitos estudados na disciplina de Circuitos Digitais.
